# Update User

Update an existing user's information.

<RequestExample>

```bash cURL
curl -X PUT "https://api.example.com/v1/users/user_123" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Smith",
    "metadata": {
      "company": "New Corp"
    }
  }'
```

```javascript Node.js
const response = await fetch('https://api.example.com/v1/users/user_123', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'John Smith',
    metadata: {
      company: 'New Corp'
    }
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.put(
    'https://api.example.com/v1/users/user_123',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'name': 'John Smith',
        'metadata': {
            'company': 'New Corp'
        }
    }
)

data = response.json()
```

</RequestExample>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the user to update
</ParamField>

## Request Body

<ParamField body="name" type="string">
  Updated name of the user
</ParamField>

<ParamField body="email" type="string">
  Updated email address
</ParamField>

<ParamField body="role" type="string">
  Updated user role (one of: 'user', 'admin')
</ParamField>

<ParamField body="metadata" type="object">
  Updated metadata for the user
  <Expandable title="properties">
    <ParamField body="company" type="string">
      Updated company name
    </ParamField>
    <ParamField body="phone" type="string">
      Updated phone number
    </ParamField>
  </Expandable>
</ParamField>

## Response
