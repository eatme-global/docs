# Delete User

Permanently delete a user and all associated data.

<RequestExample>

```bash cURL
curl -X DELETE "https://api.example.com/v1/users/user_123" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript Node.js
const response = await fetch('https://api.example.com/v1/users/user_123', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
    'https://api.example.com/v1/users/user_123',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

data = response.json()
```

</RequestExample>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the user to delete
</ParamField>

## Response

<ResponseField name="data" type="object">
  Empty object indicating successful deletion
</ResponseField>

<ResponseExample>

```json Response
{
  "data": {
    "deleted": true,
    "id": "user_123"
  }
}
```

</ResponseExample>

## Error Codes

<ResponseField name="400" type="object">
  Invalid user ID format
</ResponseField>

<ResponseField name="401" type="object">
  Invalid or missing API key
</ResponseField>

<ResponseField name="403" type="object">
  Insufficient permissions
</ResponseField>

<ResponseField name="404" type="object">
  User not found
</ResponseField>

<Note>
  This action cannot be undone. Make sure to back up any important data before deletion.
</Note>
