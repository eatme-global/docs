# Adyen Restaurant Setup Guide

This guide outlines the process of setting up Adyen payment processing for a new restaurant in DineTap.

## Prerequisites

- Adyen merchant account
- Business documentation
- Bank account information
- Terminal requirements list
- PCI compliance documentation

## Setup Process

<Steps>
  <Step title="Merchant Account Setup">
    1. Access Adyen Customer Area
    2. Create new merchant account
    3. Complete company information
    4. Submit verification documents
  </Step>

  <Step title="DineTap Integration">
    1. Get API credentials from Adyen
    2. Configure in DineTap admin panel
    3. Set up webhook notifications
    4. Configure payment methods
  </Step>

  <Step title="Terminal Setup">
    1. Order Adyen terminals
    2. Configure terminal settings
    3. Link to merchant account
    4. Install terminal application
  </Step>

  <Step title="Payment Configuration">
    Enable required payment methods:
    - Card present transactions
    - Contactless payments
    - Mobile wallets
    - QR code payments
  </Step>
</Steps>

## Account Configuration

### API Credentials
- API key
- Client key
- Merchant account ID
- Live endpoint prefix
- Webhook hmac key

### Terminal Settings
- Terminal model selection
- Network configuration
- Receipt customization
- Offline mode settings

## Testing Process

<Steps>
  <Step title="Test Environment Setup">
    1. Configure test credentials
    2. Set up test terminal
    3. Enable test payment methods
  </Step>

  <Step title="Transaction Testing">
    Verify the following:
    - Card payments
    - Contactless transactions
    - Refunds
    - Partial authorizations
  </Step>
</Steps>

## Troubleshooting Guide

### Common Issues
- Terminal connection problems
- Payment declined errors
- Webhook delivery failures
- Settlement delays
