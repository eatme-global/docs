# POS System Setup

## Initial Login

1. **Access the System**
   - Launch the POS application
   - Enter your admin credentials
   - Complete two-factor authentication (if enabled)

2. **Business Information**
   - Enter your business name
   - Add business address
   - Configure tax information
   - Set up operating hours

## Hardware Setup

### Receipt Printer
- Connect your receipt printer
- Install necessary drivers
- Test print functionality
- Configure paper settings

### Cash Drawer
- Connect the cash drawer
- Test opening mechanism
- Configure auto-open settings

### Card Reader
- Connect payment terminal
- Configure payment processor
- Test transaction flow
- Set up receipt preferences

## Staff Management

1. **Create User Accounts**
   - Add manager accounts
   - Set up staff profiles
   - Configure access levels
   - Set PIN codes/passwords

2. **Roles and Permissions**
   - Define role hierarchy
   - Set access restrictions
   - Configure override permissions

## Menu Configuration

1. **Basic Setup**
   - Create categories
   - Add initial items
   - Set prices
   - Configure modifiers

2. **Advanced Options**
   - Set up combo meals
   - Configure special pricing
   - Add item images
   - Set inventory tracking

## Testing

Before going live:
- Process test transactions
- Print sample receipts
- Test void operations
- Verify end-of-day procedures

## Next Steps

After completing setup:
- Train your staff
- Review [configuration options](/pos/getting-started/configuration)
- Set up [menu items](/pos/guides/menu-setup)
- Learn about [daily operations](/pos/guides/daily-operations)
